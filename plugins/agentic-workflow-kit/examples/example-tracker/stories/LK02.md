## Scope boundary

**In scope**

- Resolve short code to original URL.
- Return a permanent redirect for known codes.

**Out of scope**

- Click counting; see LK03.
- Link creation; handled by LK01.

## Canonical impact

Establishes the redirect HTTP contract (301 response shape, not-found behavior) — a product behavior decision that LK04 must fold into canonical docs.

## Candidate surfaces

- **Files/modules:** redirect route/handler, link store lookup.
- **Queries/schema:** read by short code.
- **Prompts/tools:** none expected.
- **Events/metrics:** optional redirect audit event.
- **Components/routes:** short-code redirect route.

## Validation expectations

- Known short codes redirect with 301.
- Unknown short codes return a bounded not-found response.
