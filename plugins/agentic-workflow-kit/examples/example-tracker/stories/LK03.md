| Dependency | Reason |
| --- | --- |

## Scope boundary

**In scope**

- Increment a click count when a redirect succeeds.
- Expose a per-link count.

**Out of scope**

- Advanced analytics, referrers, geo, or time-series reporting.

## Candidate surfaces

- **Files/modules:** redirect handler, link store update, count display/API.
- **Queries/schema:** click count field or event table.
- **Prompts/tools:** none expected.
- **Events/metrics:** redirect-count metric.
- **Components/routes:** count display or API.

## Validation expectations

- Redirecting increments the count once.
- Reading the count returns the current value.
