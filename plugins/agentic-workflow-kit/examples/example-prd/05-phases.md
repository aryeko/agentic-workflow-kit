# Phases

## Why phases
Ship a usable shortener first; add analytics once redirection is solid.

## Phase breakdown
### Phase 0 — Shorten and redirect
- **Goal:** create a code and resolve it.
- **Scope:** link creation, redirect.
- **Exit bar:** a created code redirects to its destination.

### Phase 1 — Analytics
- **Goal:** owners see usage.
- **Scope:** per-link click counts.
- **Exit bar:** the count increments on each visit.
