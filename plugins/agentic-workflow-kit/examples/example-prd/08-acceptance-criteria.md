# Acceptance criteria

## Links (PREFIX `L`)

| # | Criterion | Designation |
| --- | --- | --- |
| **L-1** | An owner can submit a long URL and receive a unique short code. | **[ship blocker]** |
| **L-2** | Visiting a short code redirects (HTTP 301) to the original URL. | **[ship blocker]** |

## Analytics (PREFIX `A`)

| # | Criterion | Designation |
| --- | --- | --- |
| **A-1** | An owner can view a click count for each of their links. | **[target]** |

## Ship-blocker summary
All `[ship blocker]` criteria (L-1, L-2) must be met before Linkly ships. `[target]`
criteria (A-1) may be deferred when a documented workaround exists.
