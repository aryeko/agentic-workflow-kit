# Success metrics

## North-star
Weekly successful redirects.

## Supporting metrics
| Metric | Target |
| --- | --- |
| Median create-to-first-click time | < 1 day |
| Redirect error rate | < 0.1% |

## Anti-metrics
We do not optimize for time-on-site; a fast bounce to the destination is the goal.
