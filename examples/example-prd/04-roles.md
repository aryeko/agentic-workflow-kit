# Roles

## Personas
- **Owner** — creates links and checks how often they are visited.
- **Visitor** — clicks a short link and is redirected.

## Capability matrix
| Capability | Owner | Visitor |
| --- | --- | --- |
| Create a short link | yes | no |
| Be redirected by a code | yes | yes |
| View click counts | yes | no |
