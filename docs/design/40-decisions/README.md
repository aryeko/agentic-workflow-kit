This layer contains accepted architecture decisions and their rationale for kit-vnext. All decisions
are binding on domain designs; a domain that needs to revisit an accepted decision raises the
question here first.

- [Accepted decisions](accepted-decisions.md) — AD-1 through AD-20

permissible.
