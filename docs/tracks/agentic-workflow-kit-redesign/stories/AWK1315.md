## Scope boundary

**In scope**

- Add targeted tests for the thinnest-covered branches (identified via the combined coverage report) to
  lift real headroom; re-baseline the ratchet thresholds upward (no regression, keep moving toward 90).
- Fix the stale-`coverage/` local footgun: an interrupted prior `pnpm test` leaves a stale `coverage/`
  dir that crashes the next run with `ENOENT .../coverage/.tmp/coverage-0.json`. Add a pre-test clean
  (or provider config) so a re-run from the same working tree succeeds.

**Out of scope**

- Behavioral code changes (covered by AWK13.13/13.14).
- The end-to-end story-run test and live child-control coverage already landed in AWK13.11.
