# Epic Charters

Epic folders bundle each milestone's charter, story DAG, and future story contracts.

Each epic `README.md` is the epic charter: purpose, included domain charters, dependencies, outputs,
boundaries, and readiness for downstream epics. It should stay high level; dispatch-ready acceptance
criteria belong in story files.
