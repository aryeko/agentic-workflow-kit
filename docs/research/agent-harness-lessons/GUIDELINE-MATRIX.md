# Agent harness guideline matrix

This matrix maps source-derived guidelines to kit-vnext evidence and recommended disposition.

## Status legend

- `strong` - Present in design and backed by tooling, code, tests, or durable docs.
- `partial` - Present in design or scaffolding, but not yet fully implemented or verified.
- `gap` - Missing as a concrete product or repo practice.
- `defer` - Valid lesson, but premature for the current implementation stage.
